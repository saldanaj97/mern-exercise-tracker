# Exercise tracker
This is a basic workout tracking application made by following the tutorial at: https://medium.com/@beaucarnes/learn-the-mern-stack-by-building-an-exercise-tracker-mern-tutorial-59c13c1237a1 

## Purpose
The purpose of making this applcation was to continue learning how to put a frontend and backend together using the MERN tech stack. 

## Skills worked/gained
Practied: React, Node.js, RESTful APIs
Gained: MogoDB, Express
